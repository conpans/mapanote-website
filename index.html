Installation Instructions (Windows 10/11)

Follow these steps carefully to install MapaNote:

Step 1: Download the Installer

  1. Click either download button above
  2. Your browser may warn you - click "Keep" or "Download anyway"

Step 2: Restore from Windows Defender Quarantine

Windows Defender will likely delete the file automatically. To restore it:

  1. Open Windows Security (search in Start menu)
  2. Click "Virus & threat protection"
  3. Click "Protection history"
  4. Find Mapanote-0.1.0-setup.exe in the list
  5. Click on it and select "Actions" ‚Üí "Allow on device"
  6. The file will be restored to your Downloads folder

Step 3: Run the Installer

  1. Double-click the downloaded file
  2. Windows SmartScreen will show: "Windows protected your PC"
  3. Click "More info"
  4. Click "Run anyway"
  5. Follow the installation wizard

Step 4: Allow Controlled Folder Access (If Needed)

If MapaNote can't save your notes, add it to allowed apps:

  1. Open Windows Security
  2. Go to "Virus & threat protection"
  3. Click "Manage ransomware protection"
  4. Under "Controlled folder access", click "Allow an app through Controlled folder access"
  5. Click "Add an allowed app"
  6. Browse to: C:\Program Files\MapaNote\MapaNote.exe
  7. Click "Open"

Step 5: Launch MapaNote

  1. Find MapaNote in your Start menu
  2. Click to launch
  3. Start organizing your geographic notes!

Why is this so complicated?

MapaNote is unsigned software (no code signing certificate). Windows treats all unsigned apps as potential threats, even though MapaNote is completely safe.

The source code is publicly available on GitHub for verification.

Future versions (v1.0+) will be code-signed to eliminate these warnings.

  • Installation complexity: Windows security makes installation difficult (see above).
